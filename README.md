# 🔐 Password Generator

A simple command-line password generator written in Python. Instantly generate strong, random passwords with full control over length and character types.

---

## Features

- 🔡 **Uppercase & Lowercase letters**
- 🔢 **Numbers**
- 🔣 **Symbols**
- 📏 **Custom length** — you choose how long the password is

---

## How It Works

The generator builds a character pool based on your chosen options (letters, numbers, symbols) and then randomly selects characters from that pool to create a password of your desired length. Each password is randomly generated every time, making it unpredictable and secure.

---

## Usage

Run the script from the command line:

```bash
python main.py
```

Then follow the prompts:

```
How many letters would you like in your password? 5
How many symbols would you like? 2
How many numbers would you like? 3

Your password is: *1KC!I1x8r
```

---

- Python 3.x
